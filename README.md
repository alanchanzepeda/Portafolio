# Enciclopedia Interactiva: Recursos y Necesidades de México

Proyecto web educativo que presenta información sobre los recursos naturales y las necesidades económicas de México mediante una interfaz interactiva inspirada en enciclopedias digitales.

## Autor
Alan Sauriel

## Descripción
Esta página web funciona como un portafolio virtual de evidencias para la materia **Recursos y necesidades de México**.

La aplicación permite explorar diferentes temas como:

- Recursos naturales
- Problemas ambientales
- Población
- Sectores económicos
- Educación

El objetivo es presentar la información de forma clara, visual e interactiva.

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
